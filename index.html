Typography

In order to keep typographic classes as reusable as possible, there are no padding/margin attributes added. Either use a helper class if the element is standalone or, if the type is used within a repeating block__element pattern, use the spacing measures tied to the __element class.

NB* This is a work in progress. Some classes may not yet be available.
